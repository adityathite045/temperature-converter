# Web Projects

This repository contains beginner-level web applications developed to strengthen my fundamentals in HTML, CSS, and JavaScript. Each project focuses on core logic, input validation, and clean user interface design.

## Projects

### 1. Temperature Converter
A web-based temperature converter that allows users to convert values between Celsius and Fahrenheit using JavaScript. The project focuses on accurate calculations and a simple, user-friendly interface.

*Technologies:* HTML, CSS, JavaScript

---

### 2. Percentage Calculator
A percentage calculator that accepts subject-wise marks, validates inputs within the range of 0–100, calculates the total and percentage, and displays results in a structured table format.

*Technologies:* HTML, CSS, JavaScript

---

### 3. Voting Eligibility Checker
A web application that accepts a user’s age, validates the input, and determines voting eligibility using conditional logic with clear visual feedback.

*Technologies:* HTML, CSS, JavaScript

---

### 4. Attendance Calculator
A calculator that takes total and present student counts, validates inputs, and computes attendance percentage with user-friendly output.

*Technologies:* HTML, CSS, JavaScript

---

## Purpose
These projects were developed as part of my learning journey in web development and to demonstrate basic problem-solving, logical thinking, and programming skills.
